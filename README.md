# socceHR-dashboard
This Repository holds the Supplemental Material to the socceHR dashboard submission.

1. 'high-fidelity prototype - interview questions.pdf' includes both the interview questions for the usability interviews, as well as the ten tasks the domain experts had to undertake.
2. The low-fidelity prototypes include figma designs (in PDF-fromat) which were shown to the domain experts:
     - Prototype 1: 'protype_1.pdf'
     - Prototype 2: 'protype_2.pdf'
     - Prototype 3: 'protype_3.pdf'
3. The socceHR Demo-video demonstrates all interactions and views of the dashboard:
     - socceHR video.mov: demo video
     - captions-socceHR-video.sbv: subtitles for accessibility
